## 🛠 How It Works

### Wheat Yield Prediction

The app uses a machine learning technique called Random Forest to predict wheat yields based on regional data you provide. This method uses many decision trees to improve accuracy. You don’t have to create or train any models yourself. The app handles this automatically once you upload your data.

### Plant Image Analysis

For pictures of wheat plants, the app applies DeepLabV3+, a tool for segmenting images. This helps identify parts of the plant, such as leaves, stems, and heads. Segmentation means the app marks each part clearly, making it easier to study plant health.
